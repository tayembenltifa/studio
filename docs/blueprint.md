# **App Name**: Le Livre Local

## Core Features:

- Book Listing: Allow sellers to list books with details like title, author, condition, and price.
- Book Search and Filter: Enable users to search for books using keywords and filter based on categories and price range.
- User Messaging: Implement a messaging system for buyers and sellers to communicate regarding book sales.
- User Authentication: Allow users to sign up and log in using Google or Facebook accounts.
- Seller Account Management: Provide sellers with tools to manage their listings, track inquiries, and update account information.
- Admin Dashboard: Provide an admin view to oversee data, messages and users.
- Automatic Delisting: Automatically remove sold books from the site.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and knowledge.
- Background color: Light blue-gray (#ECEFF1), creating a calm, neutral backdrop.
- Accent color: Soft orange (#FFAB40) to highlight key interactive elements.
- Body font: 'PT Sans', sans-serif.
- Headline font: 'Literata', serif.
- Use clear, readable icons to illustrate book genres.
- Clean and simple grid-based layout to ensure easy browsing.
- Subtle page transitions to provide a smooth user experience.
